# Crypto Price Alert Bot

This project checks crypto prices.
It sends an alert when a coin price changes.
You can use it for Bitcoin or Ethereum.

This project is made for beginners.
All code is simple and easy to understand.

## Features
- Check live crypto prices
- Set high and low price alerts
- Support for multiple coins
- Simple command line interface

## Requirements
- Node.js installed
- Internet connection
- Free API from a crypto price website

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add your API key inside .env
6. Run: node bot.js

## Example
Enter coin name and target price.
The bot will check every minute.
It will alert when price is reached.

## Learning Goals
- Learn how to call crypto APIs
- Learn how to work with timers
- Learn how to build simple bots
